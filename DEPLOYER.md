# DEPLOYER TA MAIN COURANTE SUR TON IPHONE

## Etape 1 — Creer le repo GitHub

1. Va sur https://github.com/new
2. Nom du repo : `main-courante`
3. Coche "Public"
4. Clique "Create repository"

## Etape 2 — Uploader les fichiers

1. Sur la page du repo, clique "uploading an existing file"
2. Glisse les 5 fichiers du dossier :
   - index.html
   - manifest.json
   - sw.js
   - icon-192.png
   - icon-512.png
3. Clique "Commit changes"

## Etape 3 — Activer GitHub Pages

1. Va dans Settings (engrenage) du repo
2. Menu gauche → Pages
3. Source : "Deploy from a branch"
4. Branch : main / root
5. Clique Save
6. Attends 1-2 minutes

## Etape 4 — Installer sur ton iPhone

1. Ouvre Safari sur ton iPhone
2. Va sur : https://mokhtarischam.github.io/main-courante/
3. Appuie sur le bouton partage (carre avec fleche)
4. Choisis "Sur l'ecran d'accueil"
5. Tu as ton icone MCI — ca marche meme hors ligne
